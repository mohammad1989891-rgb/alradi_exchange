# الراضي للصرافة والحوالات 💱

تطبيق ويب متكامل لإدارة عمليات الصرافة والحوالات المالية.

## ✨ المميزات

- 🏦 **إدارة الصناديق**: إدارة أرصدة العملات المتعددة
- 💰 **عمليات الصرف**: تسجيل عمليات شراء وبيع العملات
- 👥 **إدارة الحسابات**: متابعة حسابات العملاء
- 📊 **الديون**: إدارة الديون المستحقة (لنا/علينا)
- 📈 **التقارير**: تقارير مالية تفصيلية
- 🌙 **الوضع الليلي**: دعم كامل للوضع الداكن
- 📱 **PWA**: يعمل كتطبيق مستقل على الهاتف

## 🛠️ التقنيات المستخدمة

- **Next.js 16** - إطار العمل
- **TypeScript** - لغة البرمجة
- **Tailwind CSS** - التصميم
- **shadcn/ui** - مكونات الواجهة
- **Dexie (IndexedDB)** - قاعدة البيانات المحلية
- **Framer Motion** - الحركات والانتقالات
- **Zustand** - إدارة الحالة

## 🚀 البدء

```bash
# تثبيت المتطلبات
npm install

# تشغيل التطبيق
npm run dev
```

## 📦 النشر على Vercel

1. ارفع الكود إلى GitHub
2. اذهب إلى [vercel.com](https://vercel.com)
3. اربط المستودع
4. اضغط Deploy
